# Pasta Public - Assets Estáticos

## Imagem de Fundo da Sidebar

Para usar a imagem de fundo na sidebar, salve a imagem nesta pasta com o nome:

**`sidebar-background.png`**

Ou se preferir outro formato:
- `sidebar-background.jpg`
- `sidebar-background.webp`

### Formato recomendado:
- **PNG** ou **JPG**
- **Resolução**: 256px de largura (altura pode ser maior, será cortada)
- **Orientação**: Vertical ou quadrada

### Ajustar o nome do arquivo

Se você salvar a imagem com outro nome, atualize o arquivo `app/globals.css` na linha:

```css
background-image: url('/sidebar-background.png');
```

Substitua `sidebar-background.png` pelo nome do seu arquivo.
